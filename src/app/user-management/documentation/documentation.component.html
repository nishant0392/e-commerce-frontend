N-Kart App

Highlights:
  • Built from scratch, the web application has been replicated from Flipkart.
  • Highly focused on reusability & scalability.
  • Tech Stack:
    • Front-end development: HTML5, CSS3, Bootstrap 4 & Angular 8
    • Back-end development: Node.js, MongoDB, Express.js & EJS View Engine
  • The application contains User Management, Product Management and Payment Management as well.
  • User Management provides Mobile verification.
  • Most of the Components such as navigation menu, carousel, banners, rating, etc. can easily be customized according to our needs.
  • It is supported by the backend (built from scratch) deployed at Amazon EC2 server.
